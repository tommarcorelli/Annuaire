🧱 Pare-feu

// ufw · firewalld · iptables · nftables · pf · Windows Defender

Action ufw (Debian) firewalld (RHEL) iptables nftables pf (FreeBSD) Windows
Statut / règles actives sudo ufw status verbose sudo firewall-cmd --list-all sudo iptables -L -n -v sudo nft list ruleset sudo pfctl -sr Get-NetFirewallProfile
Autoriser un port (443) sudo ufw allow 443/tcp sudo firewall-cmd --add-port=443/tcp --permanent sudo iptables -A INPUT -p tcp --dport 443 -j ACCEPT sudo nft add rule inet filter input tcp dport 443 accept pass in proto tcp to port 443 New-NetFirewallRule -DisplayName web -Direction Inbound -LocalPort 443 -Protocol TCP -Action Allow
Bloquer une IP sudo ufw deny from 203.0.113.7 sudo firewall-cmd --add-rich-rule='rule family=ipv4 source address=203.0.113.7 drop' sudo iptables -A INPUT -s 203.0.113.7 -j DROP sudo nft add rule inet filter input ip saddr 203.0.113.7 drop block in from 203.0.113.7 New-NetFirewallRule -DisplayName bloc -RemoteAddress 203.0.113.7 -Action Block
Supprimer une règle sudo ufw delete allow 443/tcp sudo firewall-cmd --remove-port=443/tcp --permanent sudo iptables -D INPUT -p tcp --dport 443 -j ACCEPT sudo nft delete rule inet filter input handle 4 éditer pf.conf Remove-NetFirewallRule -DisplayName web
Recharger / persister automatique sudo firewall-cmd --reload sudo netfilter-persistent save sudo nft list ruleset > /etc/nftables.conf sudo pfctl -f /etc/pf.conf automatique
Activer au démarrage sudo ufw enable sudo systemctl enable --now firewalld sudo systemctl enable netfilter-persistent sudo systemctl enable --now nftables sudo pfctl -e Set-NetFirewallProfile -All -Enabled True

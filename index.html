Quelques commandes en vedette

● Alpine Linux
apk add
Installe un paquet sur Alpine. Option --no-cache idéale pour Docker.
apk add --no-cache nginx
● Arch Linux
pacman -Syu
Met à jour l'intégralité du système Arch en une commande.
sudo pacman -Syu
● Windows
winget install
Installe une application via le gestionnaire de paquets natif Windows.
winget install Git.Git
● FreeBSD
pfctl
Contrôle le pare-feu PF de FreeBSD. Charge les règles depuis pf.conf.
pfctl -f /etc/pf.conf
● RHEL/Fedora
firewall-cmd
Gère firewalld sur les systèmes Red Hat avec règles permanentes.
firewall-cmd --permanent --add-service=http
● macOS
brew install
Installe n'importe quel outil CLI ou app GUI via Homebrew.
brew install --cask firefox
